# 📊 Python Data Visualization – Koneoppimiskurssi

Tämä projekti on osa koneoppimiskurssin harjoitustehtäviä.
Projektin aikana opin uusia taitoja datan visualisoinnissa ja dokumentoinnissa.

---

## ✨ Mitä tässä projektissa tehtiin?

### 📝 Osa 1 – Markdown-dokumentointi
Opettelin kirjoittamaan selkeitä ja muotoiltuja dokumentteja Markdown-kielellä.
Markdown on yksinkertainen tapa muotoilla tekstiä – sitä käytetään mm. GitHubissa ja Jupyter Notebookeissa.

### 📈 Osa 2 – Datan visualisointi Matplotlibilla
Opettelin piirtämään erilaisia kuvaajia Python-ohjelmointikielellä
Jupyter Notebook -ympäristössä.

---

## 🎓 Opitut taidot

### ✅ Markdown-syntaksi
- Otsikoiden, listojen ja taulukoiden luominen
- Kuvien ja linkkien lisääminen
- Koodiblokkien kirjoittaminen
- Dokumentin muotoilu ja rakenne

### ✅ Jupyter Notebook
- Interaktiivisen kehitysympäristön käyttö
- Python-koodin ajaminen soluissa
- Kuvaajien näyttäminen suoraan notebookissa
- Notebookin exporttaaminen eri muotoihin

### ✅ Matplotlib-kuvaajat
- Viivakuvaajat (sin ja cos)
- Usean kuvaajan piirtäminen samaan kuvaan
- Subplot-kuvaajat rinnakkain
- Eri värit ja viivalinjat
- Otsikot ja akseleiden nimet
- Scatter plot -pistekuvaajat
- Histogrammi datasta
